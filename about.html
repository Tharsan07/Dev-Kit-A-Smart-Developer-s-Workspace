About This Project

This project is a simple, interactive code editor designed for live HTML, CSS, and JavaScript coding. It allows users to write code, see real-time results, and switch between light and dark modes for a better coding experience.

Features:

  • Live HTML, CSS, and JavaScript Code Editor
  • Real-time output preview
  • Light/Dark mode toggle for comfortable coding
  • Responsive design with a clean, modern interface

This project is perfect for learning and experimenting with web development, as it provides instant feedback on code changes.
